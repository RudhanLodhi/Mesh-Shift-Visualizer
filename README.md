# Mesh Circular Shift Visualizer

Interactive visualization of circular q-shift on a 2D mesh topology.

## Features

- Input validation for p (perfect square 4-64) and q (1 to p-1)
- Stage-by-stage mesh shift visualization with directional cues
- Complexity panel comparing ring vs mesh communication steps

## Setup

```bash
npm install
npm run dev
```

## Tests

```bash
npm test -- --run
```

## Build

```bash
npm run build
npm run preview
```
